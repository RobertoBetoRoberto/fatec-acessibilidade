# Projeto base para aula de Acessibilidade

-- Botões aumentar e diminuir o tamanho da fonte --

O codigo aumenta a fonte quando clica no botão de aumentar e diminui quando clica no botão de diminuir👌

O código tem a finalidade de implementar dois botões respectivamente, com a função de aumentar e diminuir a fonte. Os botões tem a função de pegar o valor atual da fonte (--base-font) e adicionar ou remover 5 valores de tamanho (px) da fonte. 👍

!!!ALGORITIMO!!!

Clicar botão -> Botão Recebe valor atual da fonte -> Valor Fonte Aumenta/Diminui 5

-- Botão ativar e desativar auto contraste --

O codigo altera a cor da fonte principal quando clica no botão de auto contraste👌

O código tem a finalidade de implementar um botão com a função de alterar a cor da fonte para uma cor de maior destaque. O botão tem a função de pegar o valor atual da cor fonte e subistituir pela cor #fff200 e adicionar uma borda. 👍

!!!ALGORITIMO!!!

Clicar botão -> Botão adiciona/remove a class de cor de destaque da fonte e borda -> Fonte do texto principal fica em cor de destaque com bordas destacadas ou volta ao normal
